In configuration file (conf.py):

1. Any global links are added to "extlinks"
2. Any global roles are added to "rst_prolog"
3. Theme is configured in "html_theme"
4. Additional static files' location is added to "html_static_path"
5. Additional CSS file's location is added to "html_style"


In the source/_static folder:

Custom css is added as "custom.css"
